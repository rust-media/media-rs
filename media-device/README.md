A cross-platform wrapper for capture and playback devices.

## Features

- **Camera**
  - [x] **AVFoundation** (macOS/iOS)
  - [x] **Media Foundation** (Windows)
  - [ ] **V4L** (Linux)
- **Speaker/Microphone**
  - [ ] **Core Audio** (macOS/iOS)
  - [ ] **WASAPI** (Windows)
  - [ ] **ALSA** (Linux)
  - [ ] **PulseAudio** (Linux)
